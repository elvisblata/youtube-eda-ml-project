
# YouTube Data Analysis & Machine Learning Project

This project analyzes YouTube video performance using Exploratory Data Analysis (EDA) and Machine Learning.

## Objective
Predict whether a video will achieve high views based on engagement metrics.

## Models Used
- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
